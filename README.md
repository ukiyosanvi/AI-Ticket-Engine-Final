

# 🚀 AI-Powered Knowledge Engine for Smart Support

An end-to-end **RAG (Retrieval-Augmented Generation)** system designed to automate IT ticket resolution using local LLMs and vector databases.

## 🛠️ Project Overview
This project addresses the challenge of high-volume IT support tickets by providing automated, context-aware resolutions. By ingesting technical SOPs and manuals, the system ensures that responses are grounded in company-specific knowledge.

## ✨ Key Features
* **AI Resolution:** Uses **Llama 3.2** to generate high-accuracy answers.
* **Intelligent Retrieval:** Powered by **FAISS** and **Sentence-Transformers** for semantic document search.
* **Real-time Alerts:** Integrated with **Slack Webhooks** to notify support teams instantly based on AI confidence levels.
* **Admin Dashboard:** A responsive interface built with **Flask** to track KPIs, ticket volume, and identify "Knowledge Gaps."

## 🏗️ Tech Stack
* **Backend:** Python (Flask), SQLite
* **AI/ML:** Ollama, FAISS, Sentence-Transformers
* **Frontend:** HTML5, CSS3, JavaScript (Responsive Design)
* **Integration:** Slack API
